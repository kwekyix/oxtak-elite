# Oxtak Elite Product Page

Static product landing page for **Oxtak Elite**: Oxtak hardware pre-loaded with Xpress AI software.

## Files

| File | Purpose |
|------|---------|
| `index.html` | Page markup |
| `style.css` | All styles (dark/light theme) |
| `oxtak_3d_black.png` | Device image shown in light mode |
| `oxtak_3d_white.png` | Device image shown in dark mode |
| `xpressai_system_agents.png` | Hero section screenshot |
| `xpressai_web_agents.png` | Platform section screenshot |

## Style reference

Follows the Oxtak design system from `../oxtblog/docs/`.

- Fonts: Inter + Space Grotesk (Google Fonts)
- Logo: `https://app.oxtak.com/logo.svg`
- Theme toggle: IIFE script at the bottom of `index.html` — no separate JS file
